	Food Report - це веб-аплікація по роботі з звітами по продуктам. Ціль - зберігати замовлення і продукти. Сервіс виконує вибірку, внесення нових даних, маніпуляція з ним, виведення по фільтрам та підсумкових звітів. 
	Проект створюється на Asp.net Core 2.0. Для зберігання данних використовується mongoDB. Для автентифікації планується використовувати Microsoft Identity. Для роботи з базою данних використовуються патерни Unit Of Work і Repository. В планах застовування Bootstrap/Javascript/TypeScript для Front-End частини проекту.
